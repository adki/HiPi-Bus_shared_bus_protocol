\section{개요}
데이터 전송 버스(data transfer bus : DTB)는 데이터의 전송에 직접적으로 관련되는 신호선의 모임이며,
어드레스 버스(address bus : AB), 데이터 버스(data bus : DB),
그리고 상태 버스(status bus : SB)로 구성된다.
